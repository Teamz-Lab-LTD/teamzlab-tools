How the Mouse Sensitivity Converter Works

This mouse sensitivity converter translates your settings between different games using each game's unique yaw value. Every game engine converts mouse input to camera rotation using a yaw constant — a multiplier that determines how many degrees the camera turns per sensitivity unit per pixel of mouse movement. By knowing the yaw values of both the source and target game, the converter calculates the exact equivalent sensitivity so your muscle memory transfers perfectly.

Sensitivity Conversion Formulas

Target Sensitivity = (Source Sensitivity x Source Yaw) / Target Yaw

cm/360 = (360 x 2.54) / (DPI x Sensitivity x Yaw)

eDPI = DPI x In-Game Sensitivity

The cm/360 calculator measures the physical mouse distance for a full rotation. This is the universal standard used by pros to match sensitivity across any game, and is what this converter uses internally for all calculations.

Game Yaw Values Explained

Each game engine uses a specific yaw value built into its input system. Valorant uses 0.07, which means each sensitivity unit rotates the camera 0.07 degrees per pixel of mouse movement. CS2 and several Source-engine games use 0.022. Fortnite uses 0.5556, which is why Fortnite sensitivity values appear much lower than in other games for the same actual speed. Overwatch 2 and Call of Duty use 0.0066, Destiny 2 also uses 0.0066, and games on the Source engine like CS2, Apex, and Battlefield share the 0.022 yaw value. Understanding these values is key to using any mouse sensitivity converter accurately.

Convert Sensitivity Valorant to CS2 and Other Popular Games

The most common conversions are between tactical shooters. To convert sensitivity Valorant to CS2, multiply by approximately 3.18 (0.07/0.022). Converting from CS2 to Valorant means dividing by 3.18. Apex Legends uses the same yaw as CS2, so the sensitivity is identical between them. Fortnite conversions can seem confusing because its high yaw value (0.5556) means sensitivity numbers are much smaller. A Valorant sensitivity of 0.5 equals about 0.063 in Fortnite — a much smaller number for the same mouse speed. Why cm/360 Is the Universal Standard

While eDPI and raw sensitivity numbers are useful within a single game, cm/360 is the universal metric for comparing sensitivity across any game. It measures the physical distance in centimeters you need to move your mouse to complete a full 360-degree rotation in game. A player with 35 cm/360 needs to move their mouse 35 centimeters to turn around completely, regardless of which game they are playing. Most FPS pros use between 25 and 55 cm/360 — you can verify your own with the cm/360 calculator built into this tool. For more detailed DPI analysis, check our DPI calculator and eDPI calculator.

Tips for Switching Between Games

After converting your sensitivity, spend 15-30 minutes in the new game's practice range or aim trainer to verify it feels right. Mathematical conversion is precise but subjective feel can vary due to differences in field of view, mouse smoothing, acceleration, and engine-specific input handling. If the converted sensitivity feels slightly off, adjust by no more than 5-10% in either direction. Once you find your ideal cm/360 value, write it down — it becomes your universal reference point for any new game you try. Many pro players maintain the same cm/360 across all games they compete in.
