消費税計算

金額を入力するだけで、税込み・税抜き価格と消費税額を自動計算します。標準税率10%と軽減税率8%の両方に対応。内税（税込み→税抜き）・外税（税抜き→税込み）の変換が簡単にできます。

消費税計算の仕組み

消費税は、商品やサービスの購入時に課される間接税で、日本では2019年10月から標準税率10%が適用されています。同時に、食料品（酒類を除く）と新聞（定期購読、週2回以上発行）には軽減税率8%が適用されています。消費税は最終消費者が負担し、事業者が税務署に納付する仕組みです。

消費税の計算方法は、税抜き価格から税込み価格を求める場合（外税方式）と、税込み価格から税抜き価格を求める場合（内税方式）の2通りがあります。外税方式では「税込み価格 = 税抜き価格 × (1 + 税率)」で計算し、内税方式では「税抜き価格 = 税込み価格 ÷ (1 + 税率)」で計算します。2021年4月から総額表示（税込み価格の表示）が義務化されたため、店頭では税込み価格が表示されていますが、見積書や請求書では税抜き価格と消費税を分けて記載することが一般的です。

軽減税率の対象となる「飲食料品」は、人が食用として飲食するもの全般ですが、いくつかの例外があります。酒類（アルコール度数1%以上の飲料）は標準税率10%が適用されます。また、外食やケータリングも標準税率10%です。一方、テイクアウトや宅配は軽減税率8%が適用されます。この区別は、同じ店舗でもイートインとテイクアウトで税率が異なるという状況を生み出しています。

事業者にとって重要なのがインボイス制度（適格請求書等保存方式）です。2023年10月から導入されたこの制度では、消費税の仕入税額控除を受けるために、適格請求書（インボイス）の保存が必要となりました。インボイスには、登録番号、適用税率、消費税額等が記載されている必要があります。免税事業者からの仕入れについては、経過措置として2026年9月までは80%、2029年9月までは50%の仕入税額控除が認められています。

計算式

税抜き→税込み（外税計算）:
消費税額 = 税抜き金額 × 税率
税込み金額 = 税抜き金額 + 消費税額
税込み→税抜き（内税計算）:
税抜き金額 = 税込み金額 ÷ (1 + 税率)
消費税額 = 税込み金額 − 税抜き金額

計算例

例1: 税抜き1,000円の商品（標準税率10%）
消費税額 = 1,000 × 10% = 100円。税込み金額 = 1,000 + 100 = 1,100円。

例2: 税込み1,080円の食品（軽減税率8%）
税抜き金額 = 1,080 ÷ 1.08 = 1,000円。消費税額 = 1,080 - 1,000 = 80円。

例3: 税抜き50,000円のサービス（標準税率10%）
消費税額 = 50,000 × 10% = 5,000円。税込み金額 = 55,000円。

消費税の歴史と税率の変遷

日本の消費税は1989年4月に税率3%で導入されました。その後、1997年4月に5%、2014年4月に8%、2019年10月に10%へと段階的に引き上げられています。2019年の引き上げ時には、食料品等に対する軽減税率8%が初めて導入されました。消費税は国の税収の約3割を占める重要な財源で、社会保障費（年金・医療・介護・子育て）に充てられています。

軽減税率の対象品目

軽減税率8%が適用されるのは、食品表示法に規定する食品（酒類を除く）と、週2回以上発行される定期購読の新聞です。具体的には、スーパーやコンビニで購入する食料品、飲料（アルコール飲料を除く）、テイクアウト・デリバリーの食事が対象です。一方、外食（レストラン・フードコート等での食事）、ケータリング、酒類は標準税率10%が適用されます。おまけ付き食品や食品・非食品のセット商品は、一定の要件を満たす場合のみ軽減税率が適用されます。
