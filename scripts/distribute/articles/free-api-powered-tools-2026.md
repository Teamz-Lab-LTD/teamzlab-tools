Most "free" tools online are data harvesting operations. You enter a zip code, they log your IP. You check the weather, they sell your location. I built 6 tools that run entirely in your browser using open, no-key APIs — no accounts, no tracking, no backend.

Here's what's new:

## 1. Live Weather Checker — Any City, No API Key

**[Live Weather Checker](https://tool.teamzlab.com/weather/current-weather/)**

Powered by Open-Meteo (open-source, funded by European weather agencies). Enter any city and get:

- Current temperature in °C or °F
- Feels-like temperature, humidity, wind speed + direction
- Rain/snow probability
- Weather condition (WMO codes mapped to plain English)
- Auto-detect your location via browser geolocation
- Recent city history saved locally

No API key. No signup. Updates every 10 minutes. Works for 300,000+ cities worldwide.

## 2. ZIP Code Lookup — 29 Countries

**[ZIP Code Lookup](https://tool.teamzlab.com/tools/zip-code-lookup/)**

Enter any postal code → instantly get city, state/province, country, and coordinates. Supports US, UK, Canada, Australia, Germany, France, Netherlands, New Zealand, Switzerland, and 20+ more countries.

Powered by Zippopotamus (open-source database). No key, no signup, works offline after first load.

## 3. Fake Person Generator — Up to 100 Profiles

**[Fake Person Generator](https://tool.teamzlab.com/dev/fake-person-generator/)**

Generate realistic fake person profiles for testing, prototyping, or filling demo databases. Uses the randomuser.me API.

Each profile includes: full name, email, phone, address, city, country, age, username, and a randomly generated avatar.

- Generate 1–100 profiles at once (Mockaroo's free tier maxes at 1,000 rows/month — this has no limit)
- Download as JSON or CSV
- Filter by 14 nationalities: US, UK, Canada, Australia, Germany, France, Brazil, Spain, and more
- Copy individual email/phone with one click

Free, no signup, no account.

## 4. Currency Converter — Now with 7-Day Rate Trend Chart

**[Live Currency Converter](https://tool.teamzlab.com/tools/live-currency-converter/)**

The existing currency converter now shows a **7-day exchange rate trend** below every conversion result. You can see whether the rate is rising, falling, or stable over the past week, with exact low/high/today values.

Uses the @fawazahmed0 currency API hosted on jsDelivr CDN — completely free, no key, historical data from 2024 onwards.

## 5. US Demographics Explorer — All 50 States

**[US Demographics Explorer](https://tool.teamzlab.com/us/demographics-explorer/)**

Search, sort, and compare all 50 US states by population. Shows each state's share of the total US population, national rank, estimated households, and a visual bar chart. Click any state for a detail card.

Data from US Census Bureau (2022 estimates) via DataUSA.io. Works offline with a local fallback dataset.
