Citation Styles Explained

APA 7th Edition is the standard for psychology, education, and social sciences. It uses an author-date in-text format with a hanging indent reference list. MLA 9th Edition is used in humanities and liberal arts, featuring a Works Cited page with author-page in-text citations. Chicago 17th (Notes-Bibliography) is favored in history and some humanities disciplines, using footnotes or endnotes with a bibliography. Harvard style is widely used in UK, Australian, and European universities with an author-date approach. IEEE is the standard for engineering and computer science, using numbered references in square brackets. Vancouver is used in biomedical and health sciences, also using numbered citations in the order they appear.

Tips for Accurate Citations

Always include the DOI when available, as it provides a permanent link to the source. For websites, record the access date since web content can change or disappear. When citing books, include the edition if it is not the first. For journal articles, double-check volume, issue, and page numbers against the original publication. Use the exact title as it appears on the source, including subtitle after a colon. For multiple authors, list them in the order they appear on the work, not alphabetically, since author order often indicates contribution level. This tool generates reference list entries; for in-text citations, consult each style guide for parenthetical or narrative citation rules.

When to Use Each Format

Check your assignment guidelines or target journal requirements before selecting a citation style. Most instructors specify which format to use. If no style is specified, APA is a safe default for social sciences, MLA for humanities, and IEEE for technical fields. Consistency is key: never mix citation styles within a single document.
