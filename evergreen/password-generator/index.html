What Makes a Strong Password?

A strong password is long (12+ characters), random, and includes a mix of uppercase, lowercase, numbers, and symbols. It should not contain dictionary words, personal information, or predictable patterns. This generator uses the Web Crypto API for true cryptographic randomness.

Password Strength

Entropy (bits) = Length × log₂(Character Pool Size)

Pool sizes: lowercase=26, +uppercase=52, +digits=62, +symbols=~90

12 chars with all types = ~78 bits of entropy (very strong)
