What Are Amazon A+ Content Image Sizes

Amazon A+ Content (formerly Enhanced Brand Content) allows Brand Registered sellers to add rich media modules to their product detail pages. Each module type has strict image dimension requirements that must be followed for images to display correctly. Standard A+ Content modules generally use images ranging from 135 x 135 px for small quadrant images up to 970 x 600 px for full-width header images. Premium A+ Content modules, available to select sellers, support larger formats including video at 1280 x 720 px minimum. Uploading images with incorrect dimensions can result in distorted layouts, cropped visuals, or outright rejection by the Amazon content management system.

This guide covers all 17 module types across Standard, Premium, and Brand Story tiers. Each card shows the exact pixel dimensions required, the maximum file size allowed, and a proportional preview of the aspect ratio so you can plan your creative assets accurately before production.

Standard vs Premium A+ Content Modules

Standard A+ Content is available to all Brand Registered sellers at no additional cost. It includes modules like Image and Text, Comparison Chart, Four Image and Text, Header Image, and several overlay options. Standard modules accept images up to 2 MB in JPEG or PNG format. Most standard image widths are either 970 px (full width), 300 px (sidebar or single image), or smaller sizes for multi-image grids.

Premium A+ Content (also called A++ or Brand Story) offers more advanced modules including hotspot images, carousels with multiple slides, and embedded video. Premium modules support larger file sizes up to 5 MB and higher resolution assets. The Brand Story Hero module at 1464 x 625 px provides the widest canvas available. Premium content has been shown to increase conversion rates by 5 to 15 percent compared to standard-only listings, making it worth pursuing once your brand qualifies.

How to Use the Image Size Checker

The built-in Image Size Checker below the module grid lets you validate any image before uploading to Amazon Seller Central. Select the target module from the dropdown, then drag and drop your image file or click to browse. The checker instantly compares your image dimensions against the module requirements and reports whether it passes or fails. If the dimensions do not match, you will see the exact resize needed. It also verifies that the file size is within the allowed limit for that module tier. This eliminates trial-and-error uploads and reduces the time spent fixing rejected A+ Content submissions.

For best results, always design your images at the exact required dimensions. Avoid relying on Amazon to resize or crop your images, as this often produces poor quality results. Use RGB color mode and save at 72 DPI for web display. Keep text within the safe zone (at least 50 px from edges) to ensure readability across all devices.

Tips for Optimizing A+ Content Images

Start with high-resolution source files and resize down to the exact module dimensions. Use JPEG for photographs with many colors and PNG for graphics with text, logos, or transparency. Compress images to stay under the file size limit while maintaining visual quality. Tools like TinyPNG or ImageOptim can reduce file size by 50 to 80 percent without visible quality loss. Always preview your A+ Content on both desktop and mobile before publishing, since image layouts can shift significantly between screen sizes.

Keep your visual hierarchy consistent across all modules. Use the same brand colors, fonts, and photography style throughout your A+ Content to build trust and reinforce brand identity. Comparison charts should use uniform column widths and consistent icon styles. For text overlay modules, ensure sufficient contrast between text and background imagery for readability.
