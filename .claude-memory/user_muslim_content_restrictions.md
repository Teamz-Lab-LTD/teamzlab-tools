User is Muslim. Content restrictions for all tool building:

**NEVER build tools related to:**
- Alcohol (BAC calculators, cocktail recipes, drink mixers, wine tools, beer/brewing, homebrew)
- Gambling (betting calculators, odds calculators, casino tools, lottery)
- Any other haram content

**Instead, consider Islamic-friendly alternatives:**
- Islamic tools (Quran, prayer times, zakat calculator, hijri calendar, Eid tools)
- Halal food/restaurant tools
- Finance tools (Islamic finance, halal investment screening)

**How to apply:** Before building ANY tool or suggesting categories, filter through this lens. The Cocktail/Drink hub was removed from the queue because of this.
