Alat keuangan dan produktivitas untuk pengguna Indonesia. Hitung gaji bersih setelah PPh 21 dan BPJS, cicilan KPR dengan bunga anuitas atau flat, buat faktur pajak PPN 11%, tentukan tarif freelance berdasarkan target pendapatan, atau periksa BMI dengan standar Kemenkes. Semua alat berjalan di browser Anda — data tidak dikirim ke server mana pun.

Kalkulator Gaji

Hitung gaji bersih (take-home) dari gaji kotor dengan PPh 21, BPJS Kesehatan, dan BPJS Ketenagakerjaan.

Kalkulator Tarif Freelance

Tentukan tarif per jam, harian, dan bulanan Anda berdasarkan target pendapatan tahunan dan pengeluaran.

Generator Faktur

Buat faktur pajak profesional dengan PPN 11% atau 12%, NPWP, dan banyak item. Cetak ke PDF.

Kalkulator KPR

Cek Kesiapan Coretax DJP

Alat keuangan untuk pengguna Indonesia

Koleksi alat ini dibuat khusus untuk wajib pajak, karyawan, pekerja lepas, dan pemilik rumah di Indonesia. Kalkulator gaji menggunakan tarif PPh 21 terbaru dan PTKP resmi, kalkulator KPR mendukung metode bunga anuitas (seperti bank BCA, Mandiri, BRI) dan efektif, serta generator faktur mengikuti format e-Faktur DJP dengan PPN 11% atau 12%. Semua perhitungan transparan — Anda dapat melihat rumus dan rincian.

Gratis dan pribadi

Tidak ada akun, tidak ada langganan, tidak ada pengumpulan data. Setiap alat berjalan sepenuhnya di browser Anda menggunakan JavaScript. Data gaji, NPWP, dan informasi pribadi lainnya tidak pernah meninggalkan perangkat Anda. Cocok untuk penggunaan sehari-hari atau perencanaan keuangan tahunan.

Cepat dan praktis

Setiap alat memiliki UI ringan dengan hasil instan, opsi salin dan bagikan, serta contoh angka. Dapat digunakan di desktop atau ponsel. Bookmark halaman ini untuk akses cepat ke kalkulator yang paling sering Anda butuhkan — dari perhitungan gaji bulanan hingga simulasi KPR.
