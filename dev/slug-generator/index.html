What Is a URL Slug?

A slug is the part of a URL that identifies a page in a readable way. For example, in "example.com/blog/my-first-post", the slug is "my-first-post". Good slugs are lowercase, use hyphens instead of spaces, and contain only letters, numbers, and hyphens.
