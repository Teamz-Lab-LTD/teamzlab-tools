CSS Box Shadow Syntax

Syntax: box-shadow: [inset] offset-x offset-y blur spread color;

Example: box-shadow: 5px 5px 10px 0px rgba(0,0,0,0.3);

The inset keyword is optional and creates an inner shadow instead of an outer drop shadow. All length values are in pixels.

Understanding Box Shadow Parameters

The box-shadow CSS property adds shadow effects around an element's frame. It is one of the most commonly used CSS properties for creating depth, elevation, and visual hierarchy in modern web design. Understanding each parameter gives you precise control over the shadow appearance.

The horizontal offset (offset-x) moves the shadow left or right relative to the element. Positive values push the shadow to the right, and negative values push it to the left. The vertical offset (offset-y) moves the shadow up or down, with positive values pushing it downward and negative values pulling it upward. Together, these offsets control the apparent direction of the light source casting the shadow. A shadow with offset 5px 5px appears to come from a light source in the upper-left corner.

The blur radius controls how soft or sharp the shadow appears. A value of 0 produces a hard, crisp shadow with no blur. Higher values create increasingly soft, diffused shadows. The blur is applied using a Gaussian distribution, meaning the shadow fades smoothly from opaque at the edge of the element to fully transparent at the outer edge of the blur. Most material design and modern UI shadows use blur values between 4px and 24px for subtle, realistic effects.

The spread radius expands or contracts the shadow before the blur is applied. Positive spread values make the shadow larger than the element on all sides, while negative spread values shrink the shadow. A spread of 0 means the shadow is the same size as the element before blur is applied. Spread is useful for creating shadows that extend evenly around all sides of an element, or for creating tight glows and outlines.

Shadow Color and Transparency

The shadow color determines the hue and opacity of the shadow effect. For realistic shadows, use black or dark gray with partial transparency, such as rgba(0,0,0,0.2) for a subtle shadow or rgba(0,0,0,0.5) for a more prominent one. Colored shadows using brand colors with transparency can create modern, vibrant elevation effects that are popular in contemporary UI design. The rgba format is recommended because it lets you control the alpha (opacity) channel independently from the color, making it easy to fine-tune shadow intensity without changing the color itself.

Inset Shadows

Adding the inset keyword transforms the shadow from an outer drop shadow to an inner shadow that appears inside the element's border. Inset shadows are commonly used for pressed or recessed button states, input field styling, and creating subtle depth effects within containers. They simulate the appearance of the element being pressed into the page rather than floating above it. Combining both inset and outer shadows on the same element can create sophisticated three-dimensional effects.
